Einweisung einer Hinzufügung in den Text

Wird eine Hinzufügung in den Text eingewiesen (durch Pfeil, Referenzzeichen, o. ä.), so wird durch das Element <metamark/> auf das jeweilige Einweisungszeichen hingewiesen. Das Element <metamark/> ist dabei ein leeres Element, welches innerhalb des <add>-Elements steht. 

<add><metamark/>[Hinzufügung]</add>
VORSICHT: Die genaue Ausführung des Einweisungszeichen wird nicht näher beschrieben.
