Abschließende Textstücke

Abschließende Teile, z.B. das Impressum, am Schluss der Ausgabe werden im Element <back> wiedergegeben. Darin steht ein Nachsatz im Element <trailer>, ein mögliches Impressum im Element <div type="imprint"> (s. Kap. Einleitende Textstücke; vgl. außerdem Kapitel: Anhang).
