Umgang mit Diakritika

Diakritika, die sich auf eine eindeutige moderne Entsprechung abbilden lassen, werden entsprechend normalisiert. Dies betrifft z.B. die folgenden Zeichen:

Historische Schreibung Umsetzung als
aͤ, oͤ, uͤ ä, ö, ü
ſ s
ſs, ſz ß
Ꝛ, ꝛ R, r

Diakritika, für die keine eindeutige Entsprechung gefunden werden kann (z.B. kontextbezogen ů) werden nicht normalisiert. In diesem Fall wird das entsprechende Unicode-Zeichen gesetzt.
