Besonderheiten bei den Metadaten

Die Metadaten erhalten zusätzlich zu den DTA-Metadaten-Richtlinien weitere Manuskript-spezifische Elemente.

Das Element <bibl> der Source Description erhält den @type-Wert "MAN" für Manuskripte.

Die Manuscript Description enthält bei Manuskripten das Element <handDesc>, welches die Hände des Manuskripts in jeweils einem <handNote>-Element beschreibt. Das Element <handNote> erhält dabei eine @xml:id, in welcher die jeweilige Hand eindeutig identifiziert wird. Auf diesen Identifier kann im Text mittels des @hand-Attributs verwiesen werden.
